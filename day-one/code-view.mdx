Amplication's **Code View** provides you with instant, read-only access to the production-ready code generated for your resources.

<Check>**Transparency at Your Fingertips:** Dive into Amplication's generated code to see exactly how your resources are built, ensuring quality and alignment with your standards.</Check>

## How to Access Code View

Accessing the generated code is available at the project-level:

<Steps>
    <Step title="Navigate to the Project">
        Select the specific **Project** you want to inspect from your Amplication project dashboard.
    </Step>
    <Step title="Open the Code View Tab">
        In the Navigation bar, locate and click on the **Code View** tab.
    </Step>
    <Step title="Explore the Code">
        The Code View tab will display a read-only editor with the complete generated codebase for your service. You can browse files, examine code structure, and verify implementation details.
    </Step>
</Steps>
