---
title: Create Prisma Schema
description: Creates the Prisma schema
---

## Event Name

`CreatePrismaSchema`

## Event Parameters

<ParamField body="entities" type="Entity[]">
  The entities that will be generated as models in the Prisma schema and represent the table name. By default, this param value is the entities that the user sets in the UI (Amplication dashboard).
</ParamField>
<ParamField body="dataSource" type="PrismaDataSource">
  Represents the data source type of Prisma provider (PostgreSQL, MySQL, MongoDB, etc.).

  <Expandable title="PrismaDataSource Type Definition">

  ```ts
  type PrismaDataSource = {
    name: string;
    provider: DataSourceProvider;
    urlEnv: string;
  };
  ```

  </Expandable>
</ParamField>
<ParamField body="clientGenerator" type="PrismaClientGenerator">
  A generator determines which assets are created when you run the Prisma generate command. For more information: [Prisma Generators](https://www.prisma.io/docs/concepts/components/prisma-schema/generators).

  <Expandable title="PrismaClientGenerator Type Definition">

  ```ts
  type PrismaClientGenerator = {
    name: string;
    provider: string;
    urlEnv: string;
  };
  ```

  </Expandable>
</ParamField>

```ts example
const dataSource: PrismaDataSource = {
```
